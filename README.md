# Vox

> Speak Naturally. Vox Handles the Rest.

## Overview

Vox is an AI-powered Voice Agent that enables users to automate professional communication using natural voice commands.

Instead of manually typing emails, users simply speak naturally. Vox understands the request, generates a professional email using AI, reads it back for confirmation, sends it through Gmail, and stores the interaction for future reference.

## MVP Workflow

1. Start Listening
2. Speak a command
3. Vox understands the intent
4. AI generates a professional email
5. Vox reads the email aloud
6. User confirms using voice
7. Email is sent through Gmail
8. Email is saved to history
